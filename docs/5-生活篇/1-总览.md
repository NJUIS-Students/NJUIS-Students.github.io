## 一、页面说明

这一页是生活篇的入口页，只做总览和填写框架，不直接给出未经确认的具体推荐。

这里的内容不是官方说明，也不是商业推荐。后续补充时，建议尽量写清楚“适合什么场景”“有什么注意事项”“信息什么时候更新”，方便后来同学判断是否仍然适用。

## 五、补充格式

补充吃喝玩信息时，建议统一按下面格式写，方便后续维护。

```md
- 名称 / 地点：
- 类型：
- 位置：
- 人均 / 花费预估：
- 推荐内容：
- 适合场景：
- 交通方式：
- 预计耗时：
- 返程提醒：
- 注意事项：
- 信息更新时间：
```

:::tip 补充建议
尽量写清楚“什么情况下推荐”和“什么情况下不推荐”。这比单纯写“好吃”“好玩”更有参考价值。
:::
