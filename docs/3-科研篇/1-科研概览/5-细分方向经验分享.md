# 细分方向经验分享：故事和路径集合

这一篇希望慢慢变成一个**按方向组织的经验/故事仓库**，和留学板块里的《经验与踩坑案例集》类似。

它更偏「主观体验」而不是「客观教程」：不同方向、不同路径的学长学姐，可以在这里写下自己是怎么选方向、怎么进组、过程中遇到了什么坑、最后又是怎么做决定的。

---

## 一、可以按什么方式来组织？

后续可以按「方向」来分块，比如：

- 系统 / 网络 / 操作系统 / 分布式；
- 人工智能相关方向（CV / NLP / 多模态 / Agent / 理论等）；
- 安全 / 密码学 / 区块链；
- 图形学 / 可视化 / HCI；
- 其它交叉方向（医工、金融科技等）。

每个方向下面，再挂若干个具体故事。
