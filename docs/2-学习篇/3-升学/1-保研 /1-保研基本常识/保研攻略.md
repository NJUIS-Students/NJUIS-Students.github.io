# 计算机类保研全流程指南

## 一、计算机类保研常识

### 1. 保研中最常见的名词解释
- rank ：指学分绩排名，需要提供学校的成绩单和院系的排名证明
- 强com ：选拔过程中委员会的力量>导师个人
- 弱com ：选拔过程中导师对自己录取哪位学生的话语权极大
- bar ： 夏令营入营门槛。此处一般会卡学校和学分绩排名，有论文很加分
- 候补 ：没有拿到优营，但距离优营并不遥远的同学。如果在9月份，拿到优营的同学鸽掉了，就会按排名顺序依次候补
- 鸽穿 ：指优营鸽的太多，所有候补都已入选但仍然有空余名额的现象。一般这种现象只出现在bar设置太高的学校
- 套磁 ： 学生通过邮件向教授展示兴趣和背景

###  Q1：保研和推免有什么区别？
::: tip A：
保研=接收+推免。两个阶段都完成才算保研成功。

接收，需要自己报名争取，分为夏令营接收和预推免接收。
推免，学院公示加分后的推免综合学分绩排名，基于此确定推免名单
::: 

###  Q2：夏令营和预推免有什么区别？

::: tip A：
夏令营和预推免是一前一后的两次选拔，形式类似，一般都是笔试+面试。绝大多数学校夏令营通过后无需参加预推免，但如果夏令营候补，预推
免是第二次机会，只不过名额一般会少一些。
::: 

###  Q3：NJU智科保研哪些课程算最终推免学分绩呢？
::: tip A：
基本约等于你的学位学分绩。i.e 军理、军训、英语、体育、思修+行策😰
::: 




## 二、 保研完整时间线（3月—9月）

### 📝 材料准备阶段（3-4月）
✅ 核心任务：整理申请必备材料
- 个人简历（模板参考：[zzu-resume-template](https://github.com/M0rtzz/zzu-resume-template)）
- 成绩证明、四六级证书
- 专家推荐信等其他支撑材料

::: tip 笔者tips：
笔者除了个人简历提前准备了以外，没有提前单独准备其余材料，我是在报名第一
个夏令营的时去按投递学校的要求准备的
材料。你会发现几乎所有学校报名所需材料大差不差～
:::
---

### 🔍 实验室/导师考核阶段（5-6月）
✅ 核心任务：通过面试获取认可
- 同门面试
- 导师面试
::: tip 提示：
不同实验室的要求是不一样的。考核形式大致分为两种，一种是读文献，做报告；
另一种是做科研项目，做报告。
:::
---
### 🏕️ 夏令营考核阶段（7-8月）
✅ 核心任务：完成综合考核
- 机试
- 笔试
- 面试


### 🎯 预推免考核阶段（9月）
✅ 核心任务：冲刺最终录取资格
- 机试
- 笔试
- 面试
::: tip ps：
夏令营/预推免的考核具体内容请自行查看目标院系的保研攻略
:::
---

📈 **上岸路径总结**：
材料准备 → 实验室考核 → 夏令营考核 → 预推免考核
