# 数据结构与算法

## 基本信息

课程名称：数据结构与算法

课程介绍：本课程为计算机相关专业核心课程，讲解线性表、栈与队列、树与图、排序与查找、算法复杂度与设计技巧（分治、贪心、动态规划）等内容，旨在培养数据结构建模与算法分析能力，为高阶算法与系统课程打基础。

课程画像：

- 课程类型：平台课
- 学分：3 学分
- 考核方式：考勤 + 课堂小测 + 平时代码作业 + 期中闭卷笔试 + 期末闭卷笔试

## 课程资料区

教材：
- 《数据结构（用面向对象方法与C++描述）》/殷人昆等/清华大学出版社
- 《算法设计技巧与分析》M.H.Alsuwaiyel（沙特）著，电子工业出版社，吴伟昶等译，2016

课程资料库（slides，往年卷）：[repo](https://github.com/NJUIS-Students/Resources/tree/e7eed4df5591292bd5a3c1714695953e57e876f8/%E6%95%B0%E6%8D%AE%E7%BB%93%E6%9E%84%E4%B8%8E%E7%AE%97%E6%B3%95)

## 经验分享区

难度评价：课程内容较多且要求理解和实现，整体难度适中。

作业反馈：以算法程序实现为主，注重边界情况与复杂度分析。整体不难。

考试情报：

- 常见题型包括 **选择 + 填空 + 简答题 + 手写算法代码**
- 高分策略：平时课堂小测认真完成，上课注意听讲，老师会强调一些比较重要的内容，可以适当做一些笔记，期末重点复习。ppt内容比较多，建议平时多花时间看ppt，理解每一个算法或者数据结构的实现原理。另外代码作业注意及时提交，以免错过ddl。
