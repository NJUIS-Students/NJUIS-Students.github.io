# 数字信号处理

## 基本信息

课程名称：数字信号处理

课程介绍：本课程研究如何用数学方法**对信号进行表示、变换和运算**，聚焦于对于连续和离散时间**信号**的时域与频域分析，以及对于不同**系统**性质的探究

课程画像：

- 课程类型：核心
- 学分：2 学分
- 考核方式：平时作业 + 期末闭卷考试

- 信号与系统（第二版）上册，郑君里、应启珩、杨为理著，高等教育出版社

作业参考：

- 第一次作业：考查连续与离散信号的周期性判断、系统的记忆性 / 因果性 / 线性 / 时不变性 / 稳定性五大性质判定，以及冲激信号相关的卷积运算和导数计算。
- 第二次作业：考查连续信号的分段卷积积分、离散信号的卷积和计算，以及傅里叶变换的基本性质、周期与非周期信号的频谱特性和典型信号的傅里叶变换求解。
- 第三次作业：考查傅里叶变换、拉普拉斯变换、Z 变换的综合应用，包含抽样定理计算、系统的零输入与零状态响应求解、离散系统差分方程推导及稳定性分析。


## 经验分享区

难度评价：整体课程难度**中等偏上**，数学公式和性质偏多，需要良好的微积分基础

考试情报：

- 期末考试题型为 **填空、选择、计算**，难度适中
- 对计算准确性、答题速度要求较高（往年答题时间都比较紧张），部分题目计算量较大，对熟练度要求较高

高分策略：

- 考前把课件和作业都认真复习一遍，特别是重要的公式（如 DFT、IDFT 计算公式等，注意求和的上标），避免考试时混淆

- 多刷几套**往年卷**，熟悉题型，提高计算正确率和计算速度

- 可以理解并记忆一些常见的结论，如 时域离散性对应频域周期性、对连续频谱进行等间隔采样对应于时域上的周期延拓 等，提高填空、选择题的做题速度

- 注意答题策略，遇到没头绪的题目就先跳过
