# 计算生物学

## 基本信息

课程名称：计算生物学

课程介绍：本课程介绍计算生物学的基本理论与方法，侧重利用算法与计算模型分析生物数据。

课程画像：

- 课程类型：选修（保研必修）

- 学分：2 学分

- 考核方式：平时成绩 + 汇报成绩

## 课程资料区

课程资料库： [repo](https://github.com/NJUIS-Students/Resources/tree/main/%E8%AE%A1%E7%AE%97%E7%94%9F%E7%89%A9%E5%AD%A6)


课程任务量参考：

- 2025春学期
    - 平时成绩
        - 考勤20分 + 课堂问答20分
    - 汇报成绩
        - 汇报24分 + 报告36分
        - 主题5选1，小组完成（每组4人）
        - 主题分别为**基因组学的探索和应用**、**蛋白质组学的探索和应用**、**蛋白质结构和功能预测**、**转录组学的探索和应用**与**智能健康医疗**

::: tip 提示
不同学期在平时分比例、作业数量等方面可能会有所调整，请以当学期教学安排为准。
:::

## 经验分享区

难度评价：授课内容包含生物和计算相关的基础知识，没有期末考试，但课设汇报与论文比较费时，综合评价为难度**中等**

作业反馈：
汇报需要使用PPT，每组时间6-8分钟。论文报告的话中英文都可以（中文8000字，英文7页以上），但要注意会查重和aigc。

课堂表现：
点名可能会在第二节课，课堂问答问题较少。

高分策略：
最终分数基本看项目，认真完成课设的汇报与论文。
