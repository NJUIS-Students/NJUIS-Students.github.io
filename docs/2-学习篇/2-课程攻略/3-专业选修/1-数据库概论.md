# 数据库概论

## 基本信息

课程名称：数据库概论

课程介绍：通过对相关概念和理论的学习，可以让学生掌握如何去设计、建立、操作和管理一个具体的数据库系统，从而为以后的数据库理论研究、课程实践以及信息系统开发工作建立一个良好的基础。

课程画像：

:::tip 老版培养方案（2022 级~2024 级）
- 课程类型：选修（保研必修）
- 学分：2 学分
- 考核方式：课后作业 + 课程项目 + 期末闭卷考试
:::

:::info 新版培养方案（2025 级起）
- 课程类型：选修
- 学分：1 学分
- 考核方式：等待课程第一次开设时更新
:::

## 经验分享区

- 2025春季学期
    - 难度评价：课程虽然内容繁多，但考核比较简单，侧重关键概念与核心算法的记忆与理解，整体难度不大
    - 作业反馈：课程作业共有 2 次，占总评 10% ，形式均为课程内容填空 + 简答题，期末考试中 50% 的题目会出自作业，请认真对待
    - 项目反馈：课程项目占比 20% ，2~3 人为一组完成，均为课程知识的情景应用，且需要剪辑讲解视频，难度不大
    - 考试情报：闭卷
        - 题目来源：50% 的题目来自作业， 50% 的题目来自 PPT 内容
        - 考题类型：**选择 + 填空 + 大题**，题量虽大，但比较简单，考前认真复习作业和 PPT 就可高分通过
        - 此学期复习课划过重点
