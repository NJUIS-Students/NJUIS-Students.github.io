# 离散数学

:::tip 提示
随着时代的发展与学院的不断探索，在新版（2025级起）培养方案中，此课程已被并入 **智能科学中的应用数学** ，本博客只作历史记录，参考意义有待商榷。相关课程博客请等待课程第一次开设后更新。
:::

## 基本信息

课程名称：离散数学

课程介绍：本课程是 CS 传统课程，涵盖命题逻辑、数论、函数、组合计数、集合论、代数结构、图论等八大板块，涵盖计算机科学核心领域所需的数学基础，旨在培养严密的逻辑思维与抽象建模能力，为深入学习人工智能算法与系统设计夯实理论基石。

课程画像：

- 课程类型：平台课
- 学分：4 学分
- 考核方式：平时考勤 + 期末闭卷考试

## 课程资料区

教材：屈婉玲等，《离散数学（第三版）》，高等教育出版社

课程资料库（slides，往年卷）：[repo](https://github.com/NJUIS-Students/Resources/tree/main/%E7%A6%BB%E6%95%A3%E6%95%B0%E5%AD%A6)

## 经验分享区

难度评价：讲授内容有难度，但是考核内容并不是很难，整体课程难度中等

作业反馈：课程作业形式为部分教材课后题

考试情报：

- 2024春季学期
    - 考试题型包括 **选择 + 填空 + 计算**，绝大多数为作业原题或改编
    - 高分策略：考前要把作业题目理解透、吃透
