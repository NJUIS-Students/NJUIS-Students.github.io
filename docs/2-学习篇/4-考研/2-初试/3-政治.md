# 政治

本页用于整理考研政治的备考资料、复习节奏和经验建议。

## 复习内容

- 马克思主义基本原理
- 毛泽东思想和中国特色社会主义理论体系概论
- 中国近现代史纲要
- 思想道德与法治
- 形势与政策以及当代世界经济与政治

## 建议补充

### 复习时间安排
蛋少：十一月中下旬开始，跟着徐涛核心讲案和徐涛强化课视频(可以在youtube获取免费资源)把政治认真过一遍(不能只是走马观花)，一边过一边练肖1000对应的选择题，在肖八出来之后练肖八的选择题，在肖四出来之后做肖四的选择题，并且看一下肖四的主观题答案，在最后一个周重点背包括肖四，徐涛，腿姐在内总结的重点大题(自己要归纳总结精简)。
