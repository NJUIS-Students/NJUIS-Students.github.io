# 暑研攻略

暑研（暑期科研实习）一般安排在大三结束的暑假至大四上学期，前往海外教授的实验室进行较长期的科研实习。时长通常为 2–3 个月（短期）或 6 个月左右（长期）。如果能联系到满意的学校和教授，**建议尽量争取 6 个月的长期暑研**，这样更有可能拿到强推荐信或 return offer。

---

## 一、为什么要做暑研

1. **助力留学申请：** 对于想申请美国直博或 CS 强校热门硕士项目的同学来说，暑研是获取教授推荐信或 return offer 的核心途径。尤其对于申美的同学，美国学术界和高校普遍不太认可国内教授的推荐信，一封来自美国教授的强推往往具有决定性作用。
2. **提前适应海外环境：** 深入了解和适应美国的文化、生活和科研氛围，这些方面与国内存在不小的差异，提前体验有助于未来留学时更快融入。
3. **拓展研究视野：** 接触更前沿、更广阔的研究方向，有机会产出论文成果，同时对申请材料（尤其是 SoP 的撰写）也有很大帮助。

---

## 二、本校合作项目

南大提供美国暑研奖学金项目，同学们可以关注。以下是大致的时间线：

| 时间 | 事项 |
|------|------|
| 大三上 9–10 月初 | 开放申请 |
| 10 月末 | 资料提交 + 面试 |
| 11 月 | 教授录取反馈 |
| 12 月 – 次年 2/3 月 | 与教授自行联系，走对方学院流程，获取 DS-2019（相当于官方录取函） |
| 3–6 月 | 办理美签（J-1 签证），越早预约和面签越好，存在被拒或被 check 的可能。签证拿下后即可开始租房、办信用卡、订机票等 |
| 7 月 | 一般 7 月左右前往线下暑研 |

**一些 Tips：**

- 南大合作项目的教授大多为本校校友（在美做教职），申请难度相比自己套磁会小不少。也有部分清华本科背景的老师，要求会更高一些。
- 有的老师会额外安排面试或考核，有的则直接根据材料和自我介绍做出选择。
- 选择教授时务必仔细浏览其主页和近期论文。同时关注组内是否有本科生或实习生——一方面，有些教授不习惯指导本科生，缺乏相关经验；另一方面，如果组内实习生较多，竞争强推或 return offer 的压力也会更大。
- 有具体问题欢迎联系：yinhongliu@smail.nju.edu.cn

## 三、自主套磁

如果在南大提供的暑期研究项目中没有找到与研究方向匹配的教授，可以选择**自主套磁**。所谓套磁，是指主动向海外大学的教授发送邮件，自我介绍并询问是否可以前往其课题组进行科研实习（Research Intern / RA）。实习形式通常为线下，周期一般为 2--6个月，也有少量远程形式。

整体流程与通过学校项目申请暑研基本一致，不同之处在于需要**自行寻找导师并主动联系**，而不是从学校提供的导师列表中选择。

------------------------------------------------------------------------

## 1. 如何寻找导师

### 1. 社交平台信息

在小红书、知乎等平台上，有时会有教授或实验室成员发布招收 Research Intern
/ RA 的信息。可以根据帖子中的联系方式进行申请。

### 2. 学术排名网站

可以在 https://csrankings.org上筛选美国高校，并按照研究方向查找相关教授。进入教授个人主页后，查看是否有以下信息：

-   是否招收 Research Intern / RA\
-   是否提供申请表（例如 Google Form）\
-   是否需要额外发送邮件说明申请

如果主页提供申请表，一般需要先填写表格，再发送邮件说明申请情况。

### 3. 阅读论文寻找导师

通过最近的顶会论文（如 CVPR、ICCV、NeurIPS、ICML等）找到感兴趣的研究方向，再查看论文作者所在实验室主页，确认是否接受实习生。

------------------------------------------------------------------------

## 2. 套磁邮件的基本内容

套磁邮件通常较为简短，主要包含以下信息：

**基本信息** - 来自哪所学校 - 什么专业 - 当前年级

**申请意图** - 希望在某个时间段前往课题组进行暑期 Research Intern / RA

**研究经历** - 参与过哪些科研项目 - 研究内容是什么 - 是否有论文或技术成果

**研究方向匹配** - 说明自己的研究方向与教授的某些研究工作或论文方向相匹配

**附件材料** - 在邮件中附上个人简历（CV） - 必要时附上论文、项目或GitHub 链接

邮件本质上是对简历内容的简要概括，让教授快速了解背景和研究能力，然后通过简历获取更完整的信息。

------------------------------------------------------------------------

## 3. 套磁数量建议

由于回复率通常较低，建议同时联系 **20--30 位教授**。

在选择导师时不必只局限于华人教授，可以同时联系不同国家背景的导师，以提高成功率。

------------------------------------------------------------------------

## 4. 其他建议

-   邮件标题清晰，例如：\`Application for Summer Research Intern from Nanjing University`

-   邮件内容保持简洁，一般控制在 **150--250 词**

-   提前准备材料：

    -   CV（科研经历为主）\
    -   成绩单\
    -   GitHub / Google Scholar / 个人主页\
    -   已发表或在投论文

-   注意时间节点：\
    多数实验室在 **1--3 月** 就会基本确定暑期实习名额

-   若 7--10 天没有回复，可以发送一次简短 follow‑up 邮件（这个很重要，本人就是follow-up之后才收到面试通知，最后拿到暑研机会的）

------------------------------------------------------------------------

## 5. 邮箱使用建议

建议使用 **Gmail 或 Outlook** 发送套磁邮件。

部分学校邮箱（例如`@nju.edu.cn`）在向国外高校邮箱发送邮件时，可能会因为邮件服务器信誉或过滤策略，被对方邮箱系统自动识别为垃圾邮件，从而进入**Spam / Junk** 文件夹，导致教授无法及时看到邮件。

使用 Gmail 或 Outlook 发送邮件通常具有更高的送达率。

------------------------------------------------------------------------

通过自主套磁获得暑研机会通常需要一定数量的尝试和持续沟通，如果没收到面试通知，那就是套的还不够多。
