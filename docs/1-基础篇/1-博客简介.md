# 博客简介

欢迎来到 **智科全家桶** —— 一个由南京大学智能科学与技术学院同学自发创建和维护的民间 Wiki！

## 我们是谁

智科全家桶是一个面向南京大学智能科学与技术学院（简称"智科"）全体同学的非官方知识共享平台。这里汇集了历届智科同学的学习经验、课程攻略、科研心得、职场见闻以及校园生活指南，旨在帮助每一位智科学子更好地度过大学时光。

:::tip 温馨提示
本站内容均由同学们自发撰写，仅代表个人经验与观点，不代表学院官方立场。如有疏漏或错误，欢迎指正和补充！
:::

## 本站内容导览

本站按照大学生活的各个方面进行分类，方便你快速找到所需信息：

| 板块 | 内容简介 |
| --- | --- |
| **基础篇** | 博客简介、学院全景、师资百科 —— 帮助你快速了解智科的全貌 |
| **学习篇** | 培养方案解读、各门课程攻略、升学（保研/考研/留学）经验 —— 学业路上的得力助手 |
| **科研篇** | 研究方向导航、竞赛概览 —— 带你走进科研的世界 |
| **职场篇** | 实习指南、求职经验 —— 为未来的职业发展做好准备 |
| **生活篇** | 苏州衣食住行、校园生活指南 —— 让你的大学生活更加丰富多彩 |
| **FAQ** | 新生常见问题、学习科研常见问题 —— 你的疑惑这里可能已有解答 |
| **贡献篇** | 贡献指南、贡献者名单 —— 欢迎你也来添砖加瓦 |

## 如何使用本站

- **侧边栏导航**：左侧边栏按照板块分类，点击即可展开查看各子项
- **全站搜索**：页面顶部的搜索框支持全站内容搜索，输入关键词即可快速定位
- **课程攻略**：每门课程的攻略通常包含基本信息、课程资料、经验分享三大板块，建议结合自身情况参考

## 参与贡献

智科全家桶是一个开源项目，我们非常欢迎每一位同学参与贡献！无论是补充课程攻略、分享科研经验，还是纠正已有内容中的错误，你的每一份贡献都弥足珍贵。

具体的贡献方式请参考 [贡献指南](/7-贡献篇/1-贡献指南)，你也可以直接前往我们的 [GitHub 仓库](https://github.com/NJUIS-Students/NJUIS-Students.github.io) 提交 Issue 或 Pull Request。

:::warning 免责声明
本站所有内容仅供参考，不构成任何形式的官方建议。课程信息、培养方案等可能随学院政策调整而变化，请以学院官方通知为准。
:::
